Регистры процессора:
AX - аккумулятор
BX - база
CX - счетчик
DX - данные
SI - указатель источника
DI - указатель приемника
BP - адрес базы в стеке
SP - адрес смещения в стеке

Регистры сегментов:
CS - кода
DS - данных
ES - дополнительный
FS - дополнительный
GS - дополнительный
SS - стека

IP - регистр указателя на команду

FLAGS - регистр флагов

Биты регистра флагов:
CF - флаг переноса
AF - вспомогательный флаг переноса
ZF - флаг нуля
SF - флаг знака
OF - флаг переполнения
PF - флаг четности

Способы адресации
mov ax, 2 - непосредственная
mov ax, es:0012 - прямая (например по адресу хранится слово)
mov ax, es:word_var - прямая, но переменная содержит в себе смещение 0012
mov ax, [bx] - косвенная, поместим в ax значение слова по адресу DS:BX
mov ax, [bx + 2] - значение следующего слова (2 байта - размер первого слова, пропустим)

Команды
mov ax, cs - записать адрес сегмента в ax
xchg ax, dx - обменять ax и dx значениями
push eax - добавить в стек значение
pop eax - восстановить значение из стека
сmp 10, 10 - сравнить 2 значения (чтобы использовать команды передачи управления)

Арифметические операции
add ax, dx - сложение, результат в ax
sub ax, dx - вычитание, результат в ax
mul dl - результат в ax = al * dl
div dl - результат в ah = al % dl(остаток) и al = al / dl (целая часть)
inc ax - инкрементирование на 1
dec ax - декрементирование на 1
neg ax - смена знака

Логические операции
and ax, dx - и
or ax, dx - или
xor ax ax - исключающее или (в примере обнуление)
not ax - инверсия
shr ax, 2 - сдвиг вправо (аналогично делению на 4)
shl ax, 2 - сдвиг влево (аналогично умножению на 4)

Команды передачи управления
jmp label - безусловый переход
je label - если равно (ZF = 0)
jg label - если >
jge label - если >=
jl label - если <
jle label - если <=

loop label - цикл (пока cx > 0)

call label - передача управления, при этом команда за call помещается в стек
ret label - возврат управления, с команды в стеке, помещенной туда через call

nop - ничего не делает, занимает место и время
